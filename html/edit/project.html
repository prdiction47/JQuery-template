Here you can edit the settings for your project.

Keyword:
The first, and primary keyword that you intend to start your research project with. All other keywords in this project should be related to this keyword.
Title:
A description of the project.
Language/Region:
The language/location in which to do research.
Note: after changing the region you will need to analyze keywords again to request the new, region-specific data.
Adult content:
Whether to include adult content in results.
