Project

Creating a project allows you to save sets of keywords, web-site and analysis data, all on a single theme.

This allows you to see all of your project research in a single workspace, save your project, as well as allowing you to load existing projects so that you can resume your work at a later stage.

Keyword:
The first, and primary keyword that you intend to start your research project with. All other keywords in this project should be related to this keyword.
Title:
A description of the project.
File:
The location where your project file will be saved.
